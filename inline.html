Apa yang dapat dilakukan Javascript?

1. JavaScript dapat mengubah konten HTML.

2. JavaScript dapat mengubah nilai atribut HTML.

Dalam hal ini JavaScript mengubah nilai atribut src (sumber) dari sebuah gambar.

3. JavaScript dapat mengubah gaya pada sebuah elemen HTML.

4. JavaScript dapat menyembunyikan elemen HTML.

5. JavaScript dapat menampilkan elemen HTML yang tersembunyi.
